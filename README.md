# Social Fresco

A brand strategy and social media agency website built with React, TypeScript, and Vite. Clean, modern design with a vibrant color palette.

## Features

- Responsive layout with Tailwind CSS
- Portfolio, testimonials, and contact sections
- SEO-friendly meta tags, Open Graph, and JSON-LD
- Accessible navigation and form labels

## Prerequisites

- Node.js 18+

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000).

## Build for production

```bash
npm run build
```

Output is in the `dist` folder. Preview the build:

```bash
npm run preview
```

## Deploy to GitHub Pages

1. In your repo: **Settings → Pages**.
2. Source: **GitHub Actions** (or deploy the `dist` folder from the `main` branch).
3. Push your code; the `dist` folder is generated by `npm run build`.

To use the GitHub Actions workflow, add `.github/workflows/deploy.yml` (e.g. using `peaceiris/actions-gh-pages` with `vite` build).

## Project structure

- `App.tsx` – Root component and layout
- `components/` – Navbar, Hero, Portfolio, Testimonials, Contact, Footer
- `index.html` – Entry HTML and SEO meta tags
- `vite.config.ts` – Vite configuration
